Visualize Data: Basics of Plotting



BASIC PLOT:

Learn more about plotting graphs with:

help(plot)

PARAMETERS:

plot(x, y, type, main, sub, xlab, ylab, col...)

Sample code written below using Atmospheric Carbon Dioxide data from Mauna Lao. Results shown as graph at top right. Remember to see available datasets use "data()".

data(co2) # load data

plot(co2, type = "l", main = "Mauna Loa Atmospheric CO2 Concentration", xlab = "Year", ylab = "CO2 Concentration", col="blue")

Can also plot by one or multiple variables, as shown below. Results shown

data(CO2)

plot(CO2$Plant, CO2$uptake, xlab = "Plant ID", ylab = "CO2 Uptake")


HISTOGRAMS:

Show the frequency of data, while specifyiing breaks, and x limits.

hist(CO2$uptake) # basic histogram

hist(CO2$uptake, breaks = 8) #compare with:

hist(CO2$uptake, breaks = 0:50)

hist(CO2$uptake, br= c(0,10,20,30,40,50)) # define scale/interval

hist(CO2$uptake, breaks = 10, xlim=c(20,40)) # define x limit


DENSITY: Relative Frequency

hist(CO2$uptake, freq=FALSE)
hist_freq <-hist(CO2$uptake)
plot(hist_freq$mids,100*hist_freq$counts/sum(hist_freq$counts), type="h")


STEM & LEAF:

Shows distribution of numbers by tens, hundreths, etc. Results shown

stem(CO2$uptake)




CUMULATIVE ABSOLUTE FREQUENCY:

plot(sort(CO2$uptake), (1:length(CO2$uptake)), type="s")


CUMULATIVE RELATIVE FREQUENCY:

plot(sort(CO2$uptake), (1:length(CO2$uptake))/length(CO2$uptake),type="s", ylim=c(0,1))

A simpler way to plot cumulative distribution is to use the "ecdf" function.

plot(ecdf(CO2$uptake), do.points=F, verticals=T, main = "Cumulative Distribution")




PRETTIFY:

hist(CO2$uptake, axes=FALSE, plot=TRUE, main="", xlab="", ylab="")

title(main="Histogram of CO2 Plant Uptake", col.main="blue", ylab="Frequency", xlab="CO2 Uptake", col.lab="red")


ACCESSING PARAMETERS:

The current settings for the graphics page can be accessed using the "par" function. Once you change the setting, it will remain that way until you reset it. For example, change the number of graphs shown from one to two.

par() # access current parameters

par(mfrow=c(2,1)) # change the number of rows to two

Visualize Data: Plotting Multiple Datasets



BASIC PLOT:

Plotting multiple datasets can be achieved with the basic plot() function like this:

plot(data$X, data$Indonesia, "l", col="black", ylim = range(c(0,100)));
par(new=TRUE);
plot(data$X, data$Malaysia, "l", col="blue");
par(new=TRUE);
plot(data$X, data$USA, "l", col="red");
par(new=TRUE);
plot(data$X, data$India, "l", col = "lightblue")

#OR like this:

plot(data$X, data$Indonesia, "l", col="black", ylim=range(c(1:100)));
lines(data$X, data$Malaysia, "l", col="blue");
lines(data$X, data$USA, "l", col="red");
lines(data$X, data$India, "l", col = "lightblue")


However, instead of coding multiple lines, it is much easier and efficient to use matplot().



MATPLOT: Step-by-Step

The example is created with the Urban Population data acquired from the World Health Organization. The data was manually cleaned and formatted to only reflect countries I was interested in analyzing.

First Set directory using setwd("/directory/folder/...") function and assign variable "data" as the filename.csv

Data looks like this:

Assign "x" and y" variables with each column for easier accessibility.

x <- data$X
y1 <- data$Indonesia
y2 <- data$Malaysia
y3 <- data$USA
y4 <- data$Japan
y5 <- data$Philippines
y6 <- data$India


Use the matplot function using "cbind()" to account for multiple y values. Create a legend using the "legend" function specifying the location: "topleft", size: "cex=0.4", variables, and colors.

matplot(x, cbind(y1, y2, y3, y4, y5, y6), "l", xlab = "Year", ylab = "Urban Population (%)", main = "Urban Population Prediction from 1950-2050")

legend("topleft", legend = c("Indonesia", "Malaysia", "USA", "Japan", "Philippines", "India"), col=c("black", "red", "green", "blue", "lightblue", "pink"), lty=1:2, cex=0.4)
