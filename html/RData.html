R Data: Accessing Datasets



ACCESS DATA:

Some datasets are already available on your computer, and others have to be loaded/imported.

data() # To see all the data available on your computer

data("rivers") # load data

help(rivers) # get a description of the data


READ FROM FILE:

CSV files can be read using the directory where the file is stored. After you get the files directory by right clicking and clicking on "get info".

setwd("/directory/folder")

help(read.csv) #find out more on how to read a .csv file

fileData <- read.csv("rivers.csv") #assign file to a variable
