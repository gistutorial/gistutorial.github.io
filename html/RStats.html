STATS 101: Exploring R Basics

MATHEMATICAL OPERATIONS:

An algebraic expression with numbers and operations such as addition, subtraction, multiplication, exponents, and roots can be expressed and stored in a variable.

            4 + 5 * 18
            x <- 4 + 5 * 18
            5 - 3 / 12 --> x
            x <- sqrt(36)

VECTORS:

A collection of symbols, text, or numbers. Use the "c" command, or "scan" command to easily enter data in vector format.

            x <- c(3, 6, 8)
            z <- scan() # enter the data for n numbers of data @ 1:, 2:, 3:....n:, etc.

LISTS:

A collection of objects that can be different types (string, int, etc). 

            myList <- list(name = "John", Lname = "Joe", age= 65)

Access the list using index or "$". 

            myList[3]
            # same as:
            myList$age

DATA FRAMES:

Similar to a table of numbers and strings, essentially made up of objects. Note: need to have same number of objects. 

            x <- c(1, 2, 3, 4, 5, 6)
            y <- c("a", "b", "c", "d", "e", "f")
            xy <- data.frame(numbers=x,letters=y)

Access each object by index or $. Access element by [row,column] 

            xy[3]
            xy[2]
            xy$numbers
            xy[1,4]

STATS 101: Confidence Intervals

Normal Distribution:

Finding the Confidence Interval (CI) for a normal distribution dataset with a sample mean (xbar) of 7, standard deviation (s) of 1.3, and sample size (n) of 50, using a 95% Confidence Level. 

              xbar <- 7
              s <- 1.3
              n <- 50
              # Z value * standard error
              error <- qnorm(0.975)*s/sqrt(n)
              #Lower Bound of CI
              left <- xbar - error
              #Upper Bound of CI
              right <- xbar + error
              # Population mean within interval (left, right)

T Distribution n < 30:

Finding the Confidence Interval (CI) a sample size less than 30, with a mean (xbar) of 10, standard deviation (s) of 2 and sample size (n) of 15, using a 95% Confidence Level. 

              xbar <- 10
              s <- 2
              n <- 15
              # Z value * standard error
              error <- qt(0.975, df=n-1)*s/sqrt(n)
              #Lower Bound of CI
              left <- xbar - error
              #Upper Bound of CI
              right <- xbar + error
              # Population mean within interval (left, right)
